# Lumiere Fine Dining - Restaurant Website & Admin Dashboard

A premium, fully responsive landing page for a high-end restaurant, featuring a modern UI, smooth animations, and a fully functional reservation management system. 

🌍 **Live Demo:** [https://darkodukovski.github.io/restaurant-site/](https://darkodukovski.github.io/restaurant-site/)

## 🧪 How to test the Reservation System

This project features a "mock" backend using the browser's `LocalStorage`. This means that **every user gets their own private sandbox environment** to test the booking system without interfering with other users' data!

**Try it yourself in 3 simple steps:**
1. **Make a booking:** Go to the [Live Site](https://darkodukovski.github.io/restaurant-site/) and fill out the reservation form at the bottom of the page.
2. **Access the Admin Panel:** Navigate to the [Admin Dashboard](https://darkodukovski.github.io/restaurant-site/admin.html) (`/admin.html`).
3. **Manage data:** You will see the reservation you just created! Try changing the status to "Confirmed" or "Cancelled", or use the "Delete" button to remove it.

## 💻 Tech Stack
- **HTML5 & CSS3** (CSS for maximum control)
- **Tailwind CSS** (for responsive grid layouts and utilities)
- **JavaScript** (for DOM manipulation, filtering, and LocalStorage logic)
- **Flatpickr** (for custom datetime selection)

## ✨ Features
- Mobile-first, fully responsive design
- Smooth scroll animations & sticky blur header
- Dynamic menu filtering system (Starters, Mains, Desserts, etc.)
- Front-end validation for forms
- Complete CRUD operations in the Admin Dashboard

# Lumiere Restaurant App

This is a restaurant landing page and reservation system I built. It includes a frontend for customers to view the menu and book a table and a backend with an admin panel to manage those reservations.

## Tech Stack
- Frontend: HTML, CSS (Tailwind via CDN), JavaScript
- Backend: Node.js, Express
- Database: Local JSON file (`data/reservations.json`)

## Features
- Responsive landing page with smooth scrolling and animations
- Menu section with category filtering
- Reservation form with date and time picker (Flatpickr and Choices.js)
- Admin dashboard to view, confirm, cancel or delete reservations
- Simple REST API for managing the reservation data

## Setup / How to run

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   npm start
   ```
4. Open the app in your browser:
   - Main website: `http://localhost:3000`
   - Admin panel: `http://localhost:3000/admin.html`
